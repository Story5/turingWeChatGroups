# 3.溢出操作符(`&+`,`&-`,`&*`)的理解,有没有`&/`,`&%`除法和取余的溢出操作符?

## 除零溢出

一个数除以0 i / 0，或者对0求余数 i % 0，就会产生一个错误。
```
let x = 1
let y = x / 0
```
使用它们对应的可溢出的版本的运算符&/和&%进行除0操作时就会得到0值。
```
let x = 1
let y = x &/ 0
// y 等于 0
```
